Next to each student's name, you will see present, absent, tardy buttons. Clicking on one of these marks the student as a particular registered status for today's class.

If you have already taken attendance for this class today, the latest status will be automatically marked. To change a student's status (i.e.: they arrive to class late), simply click their new status and the corresponding status will be highlighted.

Note: There is no need to refresh anymore to change a student's status.
